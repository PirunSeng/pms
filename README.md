# Welcome to PMS

## What's PMS?

It is an abbreviation for Product Mangement System.

## Requirements

	* NodeJS v8.4.0
	* MongoDB v3.4.7
	
## Getting Started

Clone the project and install dependencies

	$ git clone git@github.com:PirunSeng/pms.git
	$ cd pms
	$ npm install
	
## We're ready to go

Start the server by

	$ npm start

Navigate to `http://localhost:3000` and you're all done. Enjoy!
